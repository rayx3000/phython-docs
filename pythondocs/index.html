Introduction

Python stands as a versatile, high-level programming language intended for various purposes. Its design philosophy prioritizes the clarity of code, achieved through it use of significant indentation. Python is dynamically typed and features automatic garbage collection. It accommodates diverse programming paradigms, encompassing structured, object-oriented, and functional programming. Referred to as a "batteries included" language, Python is praised for its extensive standard library.

  • web development (server-side),
  • software development,
  • mathematics,
  • system scripting
  • Multi-paradigm language
  • Full support for object-oriented and structured programming
  • Additional support for functional and aspect-oriented programming through features like metaprogramming and metaobjects
  • Extension-based support for design by contract and logic programming
  • Dynamic typing
  • Memory management using reference counting and a cycle-detecting garbage collector
Dynamic Features:
  • Dynamic name resolution (late binding)
  • Supports functional programming with features like filter, map, reduce, list comprehensions, dictionaries, sets, and generator expressions
  • Includes modules (itertools and functools) borrowed from Haskell and Standard ML
  • Emphasis on readability, simplicity, and explicitness
  • Modular design for extensibility
  • Favors a single, obvious way of accomplishing tasks over multiple alternatives
  • Strives for a simpler, less-cluttered syntax
  • Avoids premature optimization
  • Rejects patches for marginal speed improvements at the expense of clarity
  • Fun-oriented development, reflected in its name and playful approaches in tutorials
  • "There should be one—and preferably only one—obvious way to do it" philosophy
  • Emphasis on writing code that is idiomatic, natural, and adheres to Python's minimalist philosophy
  • Use of the term "pythonic" to describe code that follows Python idioms and principles
  • Unpythonic code is characterized by being difficult to understand or resembling a transcription from another language

Python And Other Programming Languages

Python is a versatile and widely-used programming language known for its simplicity and readability. It supports both object-oriented and procedural programming paradigms. Here are some key features and aspects of Python:

  1. Readability: Python's syntax is designed to be readable and clear. This makes it an excellent choice for beginners and experienced developers alike.
  2. Dynamic Typing: Python is dynamically typed, meaning you don't need to specify the data type of a variable explicitly. This can lead to more concise code.
  3. Interpreted Language: Python is an interpreted language, which means that the Python code is executed line by line, rather than being compiled into machine code beforehand. This makes development and debugging faster.
  4. Extensive Standard Library: Python comes with a vast standard library that includes modules and packages for a wide range of tasks, from working with data structures to handling networking.
  5. Community and Ecosystem: Python has a large and active community, which contributes to a rich ecosystem of libraries and frameworks. This includes popular ones like Django and Flask for web development, NumPy and Pandas for data science, and TensorFlow and PyTorch for machine learning.
  6. Cross-Platform Compatibility: Python is available on various platforms, including Windows, macOS, and Linux, making it a versatile choice for developing applications that can run on different operating systems.
  7. Versatility: Python is used in various domains, including web development, data science, machine learning, artificial intelligence, automation, scripting, and more.

Other programming languages also have their strengths and use cases. Here are a few examples:

  1. JavaScript: Primarily used for web development to create interactive and dynamic content. It's often run in web browsers but can also be used on the server-side (Node.js).
  2. Java: A general-purpose, object-oriented language used for building enterprise-level applications, Android apps, and large-scale systems.
  3. C++: Known for its performance and used in systems/software development, game development, and applications where high performance is critical.
  4. C#: Developed by Microsoft, C# is commonly used for developing Windows applications, games using the Unity game engine, and enterprise software.
  5. Ruby: Known for its elegant syntax and used in web development, particularly with the Ruby on Rails framework.
  6. Swift: Developed by Apple, Swift is used for building iOS, macOS, watchOS, and tvOS applications
  7. Go (Golang): Developed by Google, Go is known for its simplicity and efficiency, making it suitable for systems programming and large-scale distributed systems.

Variables

In Python, variables are used to store and manipulate data. They act as placeholders that can hold different types of values, such as numbers, strings, lists, or even complex objects. Variables are assigned using the assignment operator '=', and their values can be changed throughout the program.

For example if you want store 25 in a variable 'cool' here's the code:

cool = 25
print(cool)

It will be 25 in output

For multiple variables assignment it would be like this:

first_name = "Vladimir"
last_name = "Cortez"
age = "25"

print("Hello!, " + first_name + " " + last_name)
print("You are " + age + " years old!")

Or this method instead of +:

first_name = "Vladimir"
last_name = "Cortez"
age = "25"

print(f"Hello! {first_name} {last_name})
print(f"You are {age} years old!")

Or this if you want number as int instead of strings:

first_name = "Vladimir"
last_name = "Cortez"
age = "25"

age = str(age)

print("Hello!, " + first_name + " " + last_name)
print("You are " + age + " years old!")

These are the examples of Variables in Python.

Data Types

In Python, there are several built-in data types that allow you to represent and manipulate different kinds of data. Here are some commonly used data types in Python:

  1. Numeric Types:
    • int: Represents integer values like 1, 100, or -42.
    • float: Represents floating-point numbers with decimal points like 3.14 or -0.5.
    • complex: Represents complex numbers in the form of a + bj, where a and b are floats or ints.
  2. Sequence Types:
    • str: Represents a string of characters, like "Hello, world!" or 'Python'.
    • list: Represents an ordered collection of items, enclosed in square brackets ([]).
    • tuple: Represents an ordered and immutable collection of items, enclosed in parentheses (()).
  3. Mapping Type:
    • dict: Represents a collection of key-value pairs, enclosed in curly braces ({}) or created using the dict() constructor.
  4. Set Types:
    • set: Represents an unordered collection of unique elements, enclosed in curly braces ({}) or created using the set() constructor.
    • frozenset: Represents an immutable version of a set, created using the frozenset() constructor.
  5. Boolean Type - It represents a boolean value(bool), either True or False.
  6. None - Represents the absence of a value or a null value.

These are some of the fundamental data types in Python. Each data type has its own set of operations and functions that can be performed on it. Additionally, Python allows you to create your own custom data types using classes and objects.

If Else Statement

The if-else statement is used to execute different blocks of code based on a certain condition.

Here's how it works:

  1. The if keyword is used to start the if-else statement, followed by a condition that evaluates to either True or False.
  2. If the condition is True, the code block indented under the if statement will be executed.
  3. If the condition is False, the code block indented under the else statement will be executed.

    4. Here's the example of simple If..else statement in Python

temperature = 25

if (temperature >= 30):
print("The Temperature is hot!")
else:
print("It is not hot!")

While Loops

A while loop is used to repeatedly execute a set of statements as long as a certain condition is true. The loop will continue until the condition becomes false.

Here's how it works:

  1. The while keyword is used to start the while loop, followed by a condition that evaluates to either True or False.
  2. The code block indented under the while statement will be executed repeatedly as long as the condition is true.
  3. After each iteration of the loop, the condition is checked again. If it is still true, the loop continues; otherwise, the loop terminates and the program moves on to the next statement after the loop.

Here's the example of stuck in while loop

while(0 == 0):
    print("HEY!")

Simple while loop:

count = 0

while count < 5:
print(count)
count += 1

print("Loop finished")

While loop with input

name = ""

while name == "":
name = input("Name: ")

print("Hello! " + name)
print("Program finished")

For Loops

A for loop is used to iterate over a sequence (such as a list, tuple, or string) or other iterable objects. It allows you to repeatedly execute a block of code for each item in the sequence.

Here's how it works:

  1. The for keyword is used to start the for loop, followed by a variable (often called item) that represents each item in the sequence.
  2. The in keyword is used to specify the sequence or iterable object over which the loop will iterate.
  3. The code block indented under the for statement will be executed for each item in the sequence, with the variable taking the value of each item in successive iterations.

Examples of For loop in Python:

Iterate over the list

animals = ["Dog", "Cat", "Bear"]

for animal in animals:
print(animal)

Iterate over the string

name = "Jay"

for char in name:
print(char)

Iterate over a range

for i in range(10):
    print(i)

These are the how For Loops look like and works in Python, Example #1 is iterate over the list in Python by sequence, Example #2 is iterate over the string by each letters in column sequence and Example #3 is iterate over a range in row

Nested Loops

Nested loops in Python allow you to have one loop inside another loop. This can be useful when you need to perform repetitive tasks that require multiple iterations.

Here how it works:

  1. The outer loop is defined using the for keyword, and it iterates over an outer sequence.
  2. Inside the outer loop, there is another loop defined using the for keyword, and it iterates over an inner sequence.
  3. The code block indented under the inner loop will be executed for each inner_item for each outer_item.

Examples of nested loop in Python:

Nested loops to create a multiplication table

for i in range(1, 6): # Outer loop for the rows
    for j in range(1, 11): # Inner loop for the columns
        result = i * j
        print(f"{i} x {j} = {result}")

Repeating Pattern nested loops

rows = 5
cols = 5
character = "*"

for i in range(rows):
    for j in range(cols):
        print(character, end="")
    print()
